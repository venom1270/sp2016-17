You need Visual Studio to run this website.
Tested in Visual studio 2015.

1. Open Visual Studio
2. Open this solution
3. Open NuGet Packet manager inside Visual Studio (Tools -> NuGet Package Manager -> Console)
4. Run the following command: 'Update-database'
5. Click the play button in the upper toolbar (it should have the name of you default browser)
6. Wait for the project to build and server to run. A new tab with this website will automatically open in your browser.
